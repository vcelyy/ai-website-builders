## What 10Web Does Poorly

### 1. The AI is Basic (5/10) - GPT wrapper, nothing special

**Here's what 10Web's AI actually does:** Asks you questions, feeds answers to GPT, generates WordPress site.

**The limitations:**
- No visual design AI (just picks template)
- No content intelligence (generic copy)
- No strategic thinking (templates are rigid)
- Same prompts = similar results (not diverse)
- Can't iterate easily (regenerate entire site)

**Real example:** Generated site for "plumber in Chicago." Regenerated with same prompt. Got same layout, slightly different wording. Felt like reskin, not redesign.

**The problem:** The AI doesn't "design" anything. It picks a template and fills in blanks. That's not AI design, that's template automation.

### 2. Design Quality (5/10) - Looks like WordPress

**The harsh truth:** 10Web sites look like... WordPress sites.

**What this means:**
- Template-based layouts (recognizable)
- Same themes everyone uses (Astra, OceanWP)
- Generic typography (nothing distinctive)
- Stock images everywhere (unavoidable)
- Same animations (fade-ins, sliders)

**Real example:** Showed a 10Web-generated site to a designer. They asked: "Which ThemeForest template is this?" That's not a compliment.

**Why this matters:** If you want to stand out, 10Web won't help. You're using the same templates as thousands of other sites.

### 3. You're Still Stuck with WordPress (3/10) - All the complexity remains

**The brutal truth:** 10Web generates the site, but YOU have to manage it.

**What you're still dealing with:**
- WordPress updates (core, themes, plugins)
- Plugin conflicts (fun debugging)
- Security vulnerabilities (WordPress is a target)
- Performance optimization (caching, CDNs, etc.)
- Backup management (hope you remember)
- Hosting management (if self-hosted)

**Real example:** Friend used 10Web for her business site. Loved the AI generation. Six months later:
- Plugin conflict broke site (2 hours to fix)
- WordPress update broke theme (4 hours to fix)
- Malware infection (had to hire someone to clean)
- Site slow (had to optimize caching)

**Total time spent "managing" WordPress:** 20+ hours/year
**Same site on Squarespace:** 0 hours/year

**The question:** If you're paying 10Web to generate sites with AI, why are you still doing all the WordPress maintenance?

### 4. The Editor (5/10) - Standard WordPress page builder

**10Web uses Elementor** (a popular WordPress page builder).

**The problem:** Elementor is powerful but complex.

**What makes it hard:**
- 40+ widgets (overwhelming for beginners)
- Settings everywhere (desktop, tablet, mobile)
- Styling options buried (multiple clicks to change font size)
- Performance issues (bloats the site)
- Learning curve (steeper than advertised)

**Real example:** Tried to move a button 10px left. Took 5 minutes to find the setting. In Framer? 1 second. In Squarespace? 2 seconds.

**The irony:** 10Web promises AI ease, but you're still using a complex page builder.

### 5. Price (6/10) - Adds cost on top of WordPress

**10Web pricing:**
- AI Starter: $10/month (1 site, AI generation)
- AI Ultimate: $30/month (10 sites, AI generation)
- Agency: $60/month (unlimited sites, white label)

**On top of:**

**The value math:** Only makes sense if you're already committed to WordPress.
