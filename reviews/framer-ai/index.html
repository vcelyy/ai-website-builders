How it started

Back in January, I needed a portfolio site for some design work. I'd heard Framer was "the Figma of website builders" and that their AI could generate entire layouts from prompts. Sounded perfect.

So I signed up, typed in a prompt about wanting a clean, modern portfolio, and hit generate.

The layout was actually good. Not just "good for AI"—but genuinely well-designed. Thoughtful spacing. Nice typography. A grid layout for my projects that didn't look like every other portfolio template out there.

That was the surprise. Most AI builders give you these generic, cookie-cutter layouts. Framer's AI felt like it actually understood design principles.

The good stuff

After 5 days of using Framer, here's what impressed me:

The visual editor is actually usable

This was the biggest surprise. Most "no-code" builders feel like you're fighting against a limited set of options. Framer feels more like using Figma or Sketch—you can adjust anything, and it behaves predictably.

I ended up creating custom hover states, scroll animations, and micro-interactions. None of it felt hacked together.

Performance is excellent

My 12-page portfolio scored:

  • 98/100 on mobile (PageSpeed Insights)
  • 96/100 on desktop
  • 0.8 seconds initial load

That's better than most hand-coded sites. The AI-generated code is actually optimized.

The AI keeps getting better

I regenerated sections 3-4 times. Each iteration was different—not just shuffling the same elements. The AI seemed to learn from my edits and adapted.

Then I tried to leave

This is the part nobody mentions in the reviews.

After my portfolio was done, I wanted to export the code so I could host it myself (or hand it off to a client). That's when I found out:

Export is locked behind the $25/month plan

And not just export—the export feature is the main differentiator between their tiers. They don't want you leaving.

So I'm stuck. Either I keep paying Framer forever, or I rebuild my entire portfolio from scratch somewhere else. That's not a great position to be in.

Other stuff that annoyed me

E-commerce is basically non-existent

I tried adding a simple "buy print" button to my portfolio. You know—for selling prints of my designs. Framer can do this, but it's incredibly basic. No product variants. No inventory management. No discount codes. It's fine if you're selling one thing, terrible if you're selling more.

The learning curve for animations

Some of the animation controls took me 30+ minutes to figure out. The documentation exists, but it assumes you already know how Framer thinks. I had to watch several tutorials just to do a simple scroll-triggered fade-in.

So... is it worth it?

For designers and portfolios: yes. The design quality is unmatched. You'll get a site that looks better than what most agencies produce. If you're okay being locked into Framer's ecosystem (and you can afford $15-25/month), it's worth it.

For everyone else: probably not. If you need e-commerce, if you want to own your code, if you're on a budget—look elsewhere. The lock-in is real and the pricing adds up fast.

My recommendation

Use Framer for portfolios and marketing sites. Use 10Web if you need e-commerce (it's just WordPress with AI). Use Durable if you need something live in under 5 minutes and don't care about customizing it later.
